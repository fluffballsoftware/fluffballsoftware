When writing any backend go code always make sure there is a unit test to cover the functionality. 
Make sure you review the code and the tests as a whole and don't automatically add a new test for a new line of code if adding or rewriting an existing test case is the best thing to do.
Run the unit tests and make sure they are succesfull before completing the task.
